API Documentation

{{ settings.get('sitename', 'QR Manager') }} — программный интерфейс

Public Endpoints
GET / Public
Главная страница. Перенаправляет аутентифицированных пользователей на /dashboard.
GET /login Public
Страница входа. Аутентифицированные пользователи перенаправляются на /dashboard.
POST body: username — имя пользователя
password — пароль
captcha_answer — ответ капчи (если включена)
GET /r/<key> Public
Перенаправление на целевой URL, сохранённый в QR-коде. key — уникальный идентификатор QR-кода.
Errors: 404 — ключ не найден
400 — невалидный URL в QR-коде
User Endpoints
GET /dashboard Auth
Панель управления пользователя. Отображает QR-коды, назначенные текущему пользователю.
GET /logout Auth
Выход из системы. Перенаправляет на /login.
Admin Endpoints
GET /admin/dashboard Admin
Панель управления администратора. Доступно администраторам и модераторам.
GET POST /admin/create_user Admin
Создание нового пользователя. GET — форма, POST — создание.
POST body: username, password, role, category_ids
GET POST /admin/edit_user/<id> Admin
Редактирование пользователя. GET — форма, POST — сохранение.
GET /admin/delete_user/<id> Admin
Удаление пользователя.
GET POST /admin/create_category Admin
Создание категории. GET — форма, POST — создание.
POST body: name — название категории
GET POST /admin/edit_category/<id> Admin
Редактирование категории. GET — форма, POST — сохранение.
GET /admin/delete_category/<id> Admin
Удаление категории.
GET POST /admin/generate_qr Admin
Создание нового QR-кода. GET — форма, POST — генерация.
POST body: name, link, user_id, category_ids, fill_color, back_color, scale, logo_image (file)
GET POST /admin/edit_qr/<id> Admin
Редактирование QR-кода. GET — форма, POST — сохранение.
GET /admin/download_qr/<id> Admin
Скачивание PNG-изображения QR-кода.
GET /admin/delete_qr/<id> Admin
Удаление QR-кода.
GET POST /admin/site_settings Superadmin
Настройки сайта (sitename, version). Доступно только администраторам (superadmin). GET — форма, POST — сохранение.
Error Responses
* 403 Forbidden
Доступ запрещён. Страница /error/403.
* 404 Not Found
Страница не найдена. Страница /error/404.
* 500 Internal Server Error
Внутренняя ошибка сервера. Страница /error/500.
